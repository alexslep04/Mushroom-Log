# Mushroom Log Cool team 8 group group
## Aniya Frye, Christopher Ryan, Alex Sleptsov, Benjamin Wingfield, Khushi Patel, Sriya Botlaguduru

Web application that helps users narrow down likely mushroom species from uploaded photos and with those multiple candidate species, present confidence indicators, taxonomical information, toxicity based on identification, and clear safety disclaimers to avoid dangerous consumption, while also enabling users to maintain a personal log of finds to help them keep notes themselves.
